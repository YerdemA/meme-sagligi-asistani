# Meme Sağlığı Asistanı

Meme sağlığı farkındalığını artırmak, kendi kendine muayene eğitimini yaygınlaştırmak ve kullanıcıların kişisel risk değerlendirmelerini yapmalarına yardımcı olmak amacıyla geliştirilmiş bir mobil uygulamadır.

## Özellikler

*   **Muayene Öğren:** Görsel ve sesli anlatımlarla desteklenen, adım adım kendi kendine meme muayenesi eğitimi. Erişilebilirlik odaklı büyük butonlar ve net yönergeler içerir.
*   **Risk Testi Yap:** Kullanıcının risk faktörlerini değerlendiren interaktif bir test modülü.
*   **Kullanıcı Dostu Arayüz:** Herkesin kolayca kullanabileceği, sade ve anlaşılır tasarım.
*   **Erişilebilirlik:** Görme veya okuma güçlüğü çeken kullanıcılar için büyük yazı tipleri, yüksek kontrastlı renkler ve sesli geri bildirimler.

## Proje Hakkında

Bu proje, özellikle meme kanseri farkındalığı oluşturmak ve erken teşhisin önemini vurgulamak için tasarlanmıştır. Flutter ile geliştirilmiş olup, hem Android hem de iOS platformlarında çalışabilir.

## Kurulum ve Çalıştırma

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1.  Projeyi klonlayın:
    ```bash
    git clone https://github.com/YerdemA/meme-sagligi-asistani.git
    ```

2.  Proje dizinine gidin:
    ```bash
    cd meme-sagligi-asistani
    ```

3.  Gerekli paketleri yükleyin:
    ```bash
    flutter pub get
    ```

4.  Uygulamayı çalıştırın:
    ```bash
    flutter run
    ```

## Kullanılan Teknolojiler

*   **Flutter & Dart:** Uygulama geliştirme iskeleti ve dili.
*   **audioplayers:** Sesli anlatım özellikleri için.
*   **google_fonts:** Okunabilir ve estetik yazı tipleri için.
